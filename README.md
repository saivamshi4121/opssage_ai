# OpsSage Web Platform

OpsSage is a next-generation platform for operations, SRE (Site Reliability Engineering), and DevOps teams. It provides powerful tools for monitoring, automated remediation, and AI-driven root cause analysis, enabling teams to ship with confidence and prevent incidents before they impact users.

## Features

- **Real-time Monitoring**: Sub-second visibility into distributed systems.
- **Automated Remediation**: Self-healing workflows for common infrastructure issues.
- **AI Root Cause Analysis**: Proprietary ML models that correlate logs, metrics, and traces to pinpoint failures quickly.

## Tech Stack

This project is built using modern web technologies:
- **Next.js 15+** - React Framework for production.
- **Tailwind CSS v4** - Utility-first CSS framework for rapid UI development.
- **React** - JavaScript library for building user interfaces.
- **TypeScript** - Strongly typed programming language that builds on JavaScript.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application.

## Project Structure

- `app/`: Next.js App Router containing pages and layouts.
- `components/`: Reusable React components (Navbar, Hero, FeatureCard, Footer, etc.).
- `public/`: Static assets.
